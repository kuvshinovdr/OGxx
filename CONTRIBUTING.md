# OGxx generic rules

1. Each collaborator is to be given a separate branch and assigned tasks through the issues system.
2. There are several possible activities:
   * writing code;
   * writing tests;
   * documenting the code;
   * reviewing code.

# OGxx code style

1. Snake case naming:
   * Some_type_name;
   * some_variable_or_function_name;
